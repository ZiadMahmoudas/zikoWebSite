About Me

I'm Ziad Mahmoud, a passionate full-stack web developer with a strong background in building modern, scalable, and beautiful web applications. My expertise covers Angular v20, ASP.NET, SQL Server, Bootstrap, JavaScript, TypeScript, Git, GitHub, Pusher, Electron.js, jQuery, HTML, CSS, SCSS, and Tailwind CSS.

I love transforming ideas into real-world solutions, focusing on user experience, performance, and clean code.
